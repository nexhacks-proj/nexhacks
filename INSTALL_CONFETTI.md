# Install Confetti Package

## Quick Install

After you have npm working, run:

```cmd
npm install canvas-confetti @types/canvas-confetti
```

This will install the confetti library needed for the celebration animation when swiping right.

**Note:** The app will work without this package - confetti will just be skipped gracefully. Install it to enable the celebration animation!
